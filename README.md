# ✨ Inner Light

> *Sonríe, Dios te ama. Sos lo más bonito de este mundo.*

Una página web de página única con animaciones 3D, sistema de partículas y efectos interactivos.

## 🛠 Tecnologías

| Tecnología | Uso |
|---|---|
| **HTML5** | Estructura semántica |
| **CSS3** | Animaciones, orbs, efectos visuales |
| **JavaScript (ES6+)** | Lógica e interacciones |
| **Three.js** | Escena 3D: estrellas, geometrías, luces |
| **GSAP** | Animaciones de entrada |
| **Google Fonts** | Tipografía: Cormorant Garamond + Inter |

## 📁 Estructura

```
inner-light/
├── index.html
├── css/
│   └── style.css
├── js/
│   ├── three-bg.js     ← Escena 3D (Three.js)
│   ├── particles.js    ← Sistema de partículas CSS
│   └── animations.js   ← GSAP + interacciones
└── README.md
```

## 🚀 Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/inner-light.git
   ```
2. Abre `index.html` en tu navegador (o usa Live Server en VS Code).

> **Nota:** No requiere Node.js ni instalación de dependencias. Funciona directo en el navegador.

## 🌐 GitHub Pages

Para publicarlo en GitHub Pages:
1. Ve a tu repositorio → **Settings** → **Pages**
2. En *Source*, selecciona la rama `main` y carpeta `/ (root)`
3. ¡Tu sitio estará disponible en `https://tu-usuario.github.io/inner-light/`!
